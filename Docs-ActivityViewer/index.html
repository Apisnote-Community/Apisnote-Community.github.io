APISNOTE-Activity-viewer

チームでの協働に最適な電子付箋Webツール APISNOTE で利用可能な付箋(notes)の動きを可視化するツールです。

GitHubで公開中。


付箋の動きを可視化することで各グループにおける進捗状況を一目で把握できます。

例えば下記可視化の例では,このノートブックを使っている班は 10/28 20:00 までにアイデアの再発想を終え,アイデア評価に移ったことがわかります。



Quick Start

すぐに使いたい方は「exe版の使い方」を参照


公開ファイル

ファイル 説明
aav5.exe exeファイル版APISNOTE-Activity-viewer (aav)。現在はversion5(aav5)。 pythonファイル版よりPyinstallerによりexeファイル化。
aav5.py pythonファイル版APISNOTE-Activity-viewer。ApisnoteActivityDF.py, aav_heatmap.py, aav_processmap.pyを使用。
aav5cui.py aav5に入力する内容をあらかじめpythonファイルに記載しておくことで動くaav5。
ApisnoteActivityDF.py ①APISNOTEの各シートのcsvよりデータを取得し一つのPandasデータフレームを出力する関数makeActivityArrayと　②図出力時にmakeActivityArrayより得られるDataFrameの列名を見やすくする関数xAxisIndexを格納
aav_heatmap.py makeActivityArrayから出力されるDataFrameをheatmapとして可視化する関数heatmapを格納
aav_prcosess.py makeActivityArrayから出力されるDataFrameをprocessmap(最も活動量の多いシートを示す折れ線図）として可視化する関数processmapを格納
